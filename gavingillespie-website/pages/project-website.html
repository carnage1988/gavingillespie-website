Project

Building This Website

A live portfolio project documenting how I built my own website using a custom domain, Cloudflare, GitHub, HTML, CSS and a self-hosted deployment path.

HTML CSS GitHub Cloudflare Docker Nginx

The Problem

I wanted a professional online presence that was more than just a digital CV. I wanted somewhere to document my projects, explain what I was learning and show the thought process behind the infrastructure I build.

The Goal

  • Build a clean, fast and professional portfolio website.
  • Use GitHub for source control and project history.
  • Use Cloudflare for DNS, SSL and edge protection.
  • Deploy the finished site through my own home infrastructure.

Architecture

What I Built

The first version of the site was built using plain HTML and CSS. I chose this approach so I could understand every part of the structure before introducing more complex tooling later.

I created a public GitHub repository, built the initial folder structure, created the homepage and added responsive styling.

What I Learned

  • How relative paths work in a static website.
  • How to troubleshoot missing assets using browser developer tools.
  • How to structure a small web project cleanly.
  • How to use Git add, commit and push as part of a workflow.
  • How Cloudflare sits in front of a self-hosted web service.

Next Steps

  • Deploy the website to Docker.
  • Publish it through Nginx Proxy Manager.
  • Add a downloadable CV page.
  • Add project pages for my home lab, networking and automation work.
